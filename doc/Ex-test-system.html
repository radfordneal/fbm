THE COMPUTER SYSTEM USED FOR THE EXAMPLES

The software runs mentioned in the examples were done on a Lenovo
ThinkStation P620 workstation, with a 4.0/4.3 GHz AMD Ryzen
ThreadRipper PRO 3945WX processor, with 12 cores (only one of which is
used for most of the examples), and 32 Gbytes of memory (in two 16
Gbyte DIMMs).

The operating system used was Ubuntu 20.04.2 LTS.  

The compiler used was gcc 9.3.0.  The compilation options were the
defaults in make.include.
